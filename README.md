# TGT Time

Sistema de controle de horas da TGT Studio.
Substitui o TMetric com integração Monday.com.

## Stack
- Frontend: React + Tailwind + shadcn/ui (bundled em single HTML)
- Backend: Supabase (projeto opqivuzvyvvvajokutvc)
- Deploy: Vercel (auto-deploy via GitHub)
